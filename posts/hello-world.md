안녕하세요! 이 블로그는 마크다운 파일을 그대로 브라우저에서 렌더링하는 방식으로 동작합니다.

## 이 블로그의 특징

- 프레임워크 없이 순수 HTML, CSS, JavaScript만 사용
- 마크다운 파일을 `posts/` 폴더에 넣으면 바로 글이 됩니다
- 다크모드를 지원합니다 (우상단 버튼으로 전환)
- 모바일에서도 잘 보입니다

## 글 작성 방법

새 포스트를 추가하려면:

1. `posts/` 폴더에 `.md` 파일을 만든다
2. 상단에 frontmatter를 작성한다
3. `posts/index.json`에 메타데이터를 추가한다

```json
{
  "file": "my-post.md",
  "title": "내 글 제목",
  "date": "2026-06-30",
  "description": "짧은 설명",
  "tags": ["태그"]
}
```

## 앞으로의 계획

> 좋은 글을 꾸준히 쓰는 것이 목표입니다.

앞으로 개발, 독서, 일상에 대한 글을 올릴 예정입니다. 많은 관심 부탁드립니다!
